## 🌎 Overview / Visão Geral

Shield OS is a high-performance dashboard built with **Pure JavaScript**. It simulates a security terminal where users earn XP and points through a gamified interface, while administrators have full control over the "agents" (users) via a live REST API.

> [!IMPORTANT]
> This project focuses on **Vanilla JS** mastery, handling complex DOM manipulations, state management, and API integrations without frameworks.

---

## 🚀 Features / Funcionalidades

### 🎮 Player Experience / Experiência do Jogador

- **Gamification System:** Real-time XP gain and Level-up logic.
- **Agent Shop:** Integrated store to spend points and upgrade inventory.
- **Mission Feedback:** Integrated with **EmailJS** for automated system alerts.

### 🔐 Admin Command Center / Painel de Controle (CRUD)

- **Live Monitoring:** View all active agents in the database.
- **Account Control:** Terminate (Delete) or Suspend (Block/Unblock) access instantly.
- **Inventory Audit:** Check items owned by any agent.

## ⚙️ Setup & Installation / Instalação

1. **Clone the repository / Clonar o repositório:**
   ```bash
   git clone https://github.com/RenanCesar-S/shield-os.git
   ```
2. **Environment Configuration / Configuração de Ambiente:**

   ¤ Locate env.js.example in the root folder.

   ¤ Rename it to env.js.

   ¤ Insert your MockAPI URL and EmailJS credentials.

3. **Run the project / Executar:**

   ¤ Simply open index.html using Live Server in VS Code.
